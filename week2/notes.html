Typeof

  • typeof will find out the type of a value (ie if it's boolean, string, number, object)
  • The operator is "typeof" and the string 'hello' is the operand when the statement goes like this " typeof 'hello' "
  • Example: 3 + 5. The operator is + and the numbers 3 and 5 are the operands.

Note About Variables

  • Cannot use these words for variable names:

abstract, await, boolean, break, byte, case, catch, char, class, const, continue, debugger, default, delete, do, double, else, enum, export, extends, false, final, finally, float, for, function, goto, if, implements, import, in, instanceof, int, interface, let, long, native, new, null, package, private, protected, public, return, short, static, super, switch, synchronized, this, throw, throws, transient, true, try, typeof, var, volatile, void, while, with, yield, undefined, NaN, Infinity

  • You should try to use const to declare most variables.

ParseInt (also ParseFloat)

  • parseInt converts a string back into a number. It needs a parameter of how many numbers you want it to show. (Chapter 2)

Arrays

  • In an array, when you delete a value in the array, the space is still there, its just undefined. The value returned is undefined.
  • In an array, array.length - 1 will give you the value of the last item in the array. You have to subtract one because the array starts at 0.
  • array.pop(); means that you remove the last item in the array. array.shift(); means you remove the first value of the array.
  • array.push(); means you are adding a value to the end of the array, and array.unshift() adds a value to the beginning.

Sets

  • Sets won't allow duplicates. This can be really useful in situations where you need to have unique values.
  • Sets do not have index notation for inspecting individual entries, so you can't find the value of the first element in a set.

Functions

  • There are 3 ways to write a Function. function name {console.log('Hello');}
  • You can also assign a function to a variable like this: const nameOfVariable = function nameOfFunction() {console.log ('Message');}
  • Keep your code DRY (don't repeat yourself, use functions to perform the work over and over instead of repeating yourself.).

Arrow Functions

  • const square = x => x*x;
  • const hello = () => alert('Hello World!');

Map()

  • Uses arrays to output information. For example: [1,2,3].map( square ) will give you the output of [1, 4, 9]
  • So map() essentially can take each item in an array and do something to it.
  • reduce() takes an array, and then reduces the output to just one value.
